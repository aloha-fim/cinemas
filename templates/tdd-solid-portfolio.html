The Five Principles

SOLID in Practice

S

Single Responsibility

"A class should have only one reason to change"

Each notification channel is its own class. EmailSender only changes when email logic changes. SMS changes don't affect it.

# Each sender = one job class EmailSender: def send(self, to, msg): # Only email logic class SmsSender: def send(self, to, msg): # Only SMS logic
O

Open/Closed

"Open for extension, closed for modification"

Add new channels like Slack without modifying NotificationService. Just register a new sender!

# Add Slack without touching service service.register_channel( SlackSender() ) # NotificationService unchanged! service.send("slack", "#general", "Hi")
L

Liskov Substitution

"Subtypes must be substitutable for base types"

Any NotificationSender can replace any other. The service doesn't care which sender it's using.

# All senders are interchangeable def notify(sender: NotificationSender): sender.send(to, msg) # Works with ANY sender notify(EmailSender()) notify(SmsSender()) notify(PushSender())
I

Interface Segregation

"Clients shouldn't depend on methods they don't use"

Separate protocols for BulkSender and SchedulableSender. SmsSender doesn't implement bulk (too expensive).

# Focused interfaces class BulkSender(Protocol): def send_bulk(self, ...): ... class SchedulableSender(Protocol): def schedule(self, ...): ... # EmailSender: BulkSender ✓ # PushSender: Schedulable ✓
D

Dependency Inversion

"Depend on abstractions, not concretions"

NotificationService accepts any sender via constructor injection. Easy to test with mocks!

# Inject dependencies class NotificationService: def __init__(self, senders): # Accepts ANY senders # Easy testing! service = NotificationService( senders=[MockSender()] )

        if channel == "email":
            # Email logic here

            # Push logic here

SOLID
After 
# ✅ All SOLID principles applied

class NotificationService:
    def __init__(self, senders):
        # DIP: Inject dependencies
        self._channels = {}
        for s in senders:
            self.register_channel(s)
    
    def register_channel(self, sender):
        # OCP: Extension point
        self._channels[sender.channel_name] = sender
    
    def send(self, channel, to, msg):
        # LSP: Any sender works
        return self._channels[channel].send(to, msg)
